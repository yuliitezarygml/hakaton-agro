# 🔌 Схема подключения AgriMap

## 📋 Полный список компонентов

| № | Компонент | Кол-во | Назначение |
|---|-----------|:------:|------------|
| 1 | **ESP32 DevKit V1** | 1 | Микроконтроллер (4MB Flash) |
| 2 | **DHT11** | 4 | Температура + влажность воздуха (4 зоны) |
| 3 | **Capacitive Soil Moisture Sensor v1.2** | 1 | Влажность почвы (ёмкостной) |
| 4 | **Rain Sensor Module** | 1 | Обнаружение дождя (цифровой) |
| 5 | **Резистор 10 kΩ** | 7 | 4× pull-up для DHT11 + 3× делитель для Soil |
| 6 | **Макетная плата** | 1 | Монтаж компонентов |
| 7 | **Провода Dupont** | ~30 | Соединения (M-M, M-F) |
| 8 | **Блок питания 5V/2A** | 1 | Питание через USB или VIN |

## ⚡ Схема питания

```
┌─────────────────────────────────────────────────────────────┐
│                      Питание системы                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  USB 5V ──┬──► ESP32 VIN (5V регулятор → 3.3V)            │
│           │                                                 │
│           └──► Датчики (через 3.3V пин ESP32)             │
│                                                             │
│  Альтернатива: VIN пин ──► 5V внешний источник             │
│                                                             │
└─────────────────────────────────────────────────────────────┘

⚠️ ВАЖНО: Все датчики питаются от 3.3V пина ESP32!
          Максимальный ток 3.3V: ~600mA
          DHT11 (4шт): ~4mA
          Soil sensor: ~35mA
          Rain sensor: ~15mA
          ИТОГО: ~54mA ✅ Безопасно
```

## 🌡️ DHT11 — Датчики температуры и влажности (4 зоны)

### Схема подключения одного DHT11

```
                    ┌─────────────┐
                    │   DHT11     │
                    │             │
                    │  ┌───┬───┐  │
                    │  │VCC│OUT│  │GND
                    │  └─┬─┴─┬─┘  └─┬─
                    └────┼───┼──────┼──
                         │   │      │
                  3.3V ──┘   │      └── GND
                             │
                        ┌────┴────┐
                        │ 10 kΩ  │ ← Pull-up резистор
                        └────┬────┘
                             │
                          3.3V
```

### Таблица подключения всех 4 зон

| Зона | DHT11 VCC | DHT11 DATA | DHT11 GND | Pull-up резистор |
|:----:|:---------:|:----------:|:---------:|:----------------:|
| **1** | 3.3V | **GPIO 4** | GND | 10kΩ → 3.3V |
| **2** | 3.3V | **GPIO 5** | GND | 10kΩ → 3.3V |
| **3** | 3.3V | **GPIO 16** | GND | 10kΩ → 3.3V |
| **4** | 3.3V | **GPIO 13** | GND | 10kΩ → 3.3V |

### Детальная схема для каждой зоны

```
Зона 1 (GPIO 4):
  DHT11 pin 1 (VCC)  ──► ESP32 3.3V
  DHT11 pin 2 (DATA) ──► ESP32 GPIO 4  ◄──┐
                                           ├── [10kΩ] ──► 3.3V
  DHT11 pin 4 (GND)  ──► ESP32 GND

Зона 2 (GPIO 5):
  DHT11 pin 1 (VCC)  ──► ESP32 3.3V
  DHT11 pin 2 (DATA) ──► ESP32 GPIO 5  ◄──┐
                                           ├── [10kΩ] ──► 3.3V
  DHT11 pin 4 (GND)  ──► ESP32 GND

Зона 3 (GPIO 16):
  DHT11 pin 1 (VCC)  ──► ESP32 3.3V
  DHT11 pin 2 (DATA) ──► ESP32 GPIO 16 ◄──┐
                                           ├── [10kΩ] ──► 3.3V
  DHT11 pin 4 (GND)  ──► ESP32 GND

Зона 4 (GPIO 13):
  DHT11 pin 1 (VCC)  ──► ESP32 3.3V
  DHT11 pin 2 (DATA) ──► ESP32 GPIO 13 ◄──┐
                                           ├── [10kΩ] ──► 3.3V
  DHT11 pin 4 (GND)  ──► ESP32 GND
```

**Почему нужен pull-up резистор?**
- DHT11 использует однопроводной протокол (1-Wire)
- Линия данных должна быть подтянута к HIGH (3.3V) в состоянии покоя
- Без резистора датчик не будет работать стабильно

## 🌱 Capacitive Soil Moisture Sensor v1.2 — Датчик влажности почвы

### ⚠️ Делитель напряжения (ОБЯЗАТЕЛЬНО!)

Датчик может выдавать напряжение выше 3.3V, что **повредит ESP32**!  
Используем делитель на 3 резисторах по 10kΩ.

```
   Capacitive Soil Sensor v1.2
  ┌────────────────────────────┐
  │  VCC    AOUT    GND       │
  └──┬───────┬───────┬────────┘
     │       │       │
   3.3V      │      GND
             │
             │ (аналоговый выход 0–3V, иногда выше!)
             │
        ┌────┴────┐
        │ R1 10kΩ │  ← от датчика
        └────┬────┘
             │
             ├───────────────── GPIO 35 (ADC1_CH7)
             │
        ┌────┴────┐
        │ R2 10kΩ │
        └────┬────┘
             │
        ┌────┴────┐
        │ R3 10kΩ │
        └────┬────┘
             │
            GND
```

### Формула делителя

```
V_gpio = V_aout × (R2 + R3) / (R1 + R2 + R3)
       = V_aout × 20kΩ / 30kΩ
       = V_aout × 0.667

Пример:
  Если датчик выдаёт 4.5V (опасно для ESP32!)
  → GPIO 35 получит: 4.5V × 0.667 = 3.0V ✅ Безопасно
```

### Таблица подключения

| Датчик | Подключение | Примечание |
|--------|-------------|------------|
| **VCC** | ESP32 **3.3V** | Питание датчика |
| **GND** | ESP32 **GND** | Общий минус |
| **AOUT** | → [R1 10kΩ] → **GPIO 35** | Через делитель! |
| | → [R2 10kΩ] → [R3 10kΩ] → GND | |

### Калибровка датчика почвы

После подключения нужно откалибровать значения:

1. Прошей ESP32 и открой Serial Monitor (115200 бод)
2. **Сухой замер**: держи датчик в воздухе, запиши `raw=XXXX`
3. **Мокрый замер**: опусти в стакан воды, запиши `raw=XXXX`
4. Впиши значения в `src/main.cpp`:

```cpp
#define SOIL_DRY  1550  // ← твоё значение "в воздухе"
#define SOIL_WET  500   // ← твоё значение "в воде"
```

**Типичные значения:**

| Состояние | ADC raw | Напряжение | Влажность |
|-----------|:-------:|:----------:|:---------:|
| В воздухе (сухо) | ~1550 | ~1.25V | **0%** |
| Сухая земля | ~1200 | ~0.97V | **~33%** |
| Влажная земля | ~800 | ~0.64V | **~71%** |
| В воде | ~500 | ~0.40V | **100%** |

## 🌧️ Rain Sensor Module — Датчик дождя

### Схема подключения

```
  Rain Sensor Module
  ┌──────────────────┐
  │  VCC  DO  AO GND │
  └──┬───┬───────┬───┘
     │   │       │
   3.3V  │      GND
         │
         └──────────── GPIO 34 (input-only)
```

### Таблица подключения

| Датчик | ESP32 | Примечание |
|--------|-------|------------|
| **VCC** | **3.3V** | Питание (можно 5V, но 3.3V достаточно) |
| **GND** | **GND** | Общий минус |
| **DO** | **GPIO 34** | Цифровой выход (Digital Output) |
| **AO** | — | Не используется (аналоговый выход) |

### Логика работы

```cpp
// В коде (src/main.cpp):
pinMode(RAIN_PIN, INPUT);  // GPIO 34 — input-only
bool rain = !digitalRead(RAIN_PIN);  // Инверсия!

// Логика:
// 💧 Капли на датчике → DO = LOW  → rain = true
// ☀️ Сухо            → DO = HIGH → rain = false
```

**Почему инверсия?**  
Модуль Rain Sensor выдаёт LOW при обнаружении воды (активный LOW).

## 📌 Полная распиновка ESP32 DevKit V1

```
                    ┌──────────────────┐
                    │    ESP32 DevKit   │
                    │                  │
              3.3V ─┤ 3V3          VIN├─ 5V (внешнее питание)
               EN ─┤ EN           GND├─ GND
                   ─┤ VP (GPIO36)  IO23├─
                   ─┤ VN (GPIO39)  IO22├─
    DHT11 Z1 DATA ─┤ IO34 (GPIO34) TX0├─
    DHT11 Z2 DATA ─┤ IO35 (GPIO35) RX0├─
                   ─┤ IO32         IO21├─
                   ─┤ IO33         IO19├─
                   ─┤ IO25         IO18├─
                   ─┤ IO26          IO5├─ DHT11 Z2 DATA
                   ─┤ IO27         IO17├─
    DHT11 Z4 DATA ─┤ IO14 (GPIO14) IO16├─ DHT11 Z3 DATA
                   ─┤ IO12          IO4├─ DHT11 Z1 DATA
               GND ─┤ GND           IO0├─ BOOT (WiFi reset)
    DHT11 Z4 DATA ─┤ IO13          IO2├─ LED (встроенный)
                   ─┤ SD2          IO15├─
                   ─┤ SD3          SD1├─
                   ─┤ CMD          SD0├─
              3.3V ─┤ 3V3          CLK├─
                    └──────────────────┘

Используемые пины:
  GPIO 4  ← DHT11 Зона 1
  GPIO 5  ← DHT11 Зона 2
  GPIO 13 ← DHT11 Зона 4
  GPIO 16 ← DHT11 Зона 3
  GPIO 35 ← Soil Sensor (через делитель)
  GPIO 34 ← Rain Sensor
```

### ⚠️ Критические ограничения GPIO

| GPIO | Статус | Почему нельзя использовать |
|------|--------|----------------------------|
| **6, 7, 8, 9, 10, 11** | 🔴 **ЗАПРЕЩЕНЫ** | Подключены к SPI Flash — ESP32 зависнет! |
| **34, 35, 36, 39** | 🟡 **Только вход** | Input-only, нет pull-up, нет выхода |
| **0** | 🟡 **BOOT** | Используем для сброса WiFi (удержание при старте) |
| **2** | 🟡 **LED** | Встроенный LED, можно использовать |
| **12** | 🟡 **Осторожно** | Влияет на напряжение Flash при загрузке |
| **15** | 🟡 **Осторожно** | Должен быть HIGH при загрузке |
| **4, 5, 13, 16, 17, 18, 19, 21, 22, 23** | 🟢 **Безопасны** | Можно использовать свободно |

## 🔧 Пошаговая сборка на макетной плате

### Шаг 1: Установка ESP32

```
1. Установи ESP32 DevKit V1 в центр макетной платы
2. Убедись, что пины с обеих сторон доступны
3. Подключи USB-кабель для проверки питания (LED должен загореться)
```

### Шаг 2: Шины питания

```
Красная шина (+):  3.3V пин ESP32
Синяя шина (-):    GND пин ESP32

⚠️ НЕ используй 5V для датчиков! Только 3.3V!
```

### Шаг 3: DHT11 Зона 1 (GPIO 4)

```
1. Установи DHT11 в макетную плату
2. VCC (pin 1)  → красная шина (+3.3V)
3. DATA (pin 2) → GPIO 4
4. GND (pin 4)  → синяя шина (GND)
5. Резистор 10kΩ: один конец к DATA (pin 2), другой к +3.3V
```

### Шаг 4: DHT11 Зоны 2, 3, 4

```
Повтори Шаг 3 для остальных DHT11:
  Зона 2 → GPIO 5
  Зона 3 → GPIO 16
  Зона 4 → GPIO 13

Каждому нужен свой pull-up резистор 10kΩ!
```

### Шаг 5: Soil Sensor с делителем

```
1. Датчик VCC  → +3.3V
2. Датчик GND  → GND
3. Датчик AOUT → R1 (10kΩ) → точка A
4. Точка A     → GPIO 35
5. Точка A     → R2 (10kΩ) → точка B
6. Точка B     → R3 (10kΩ) → GND

Схема делителя:
  AOUT ─[R1]─┬─ GPIO35
             │
            [R2]
             │
            [R3]
             │
            GND
```

### Шаг 6: Rain Sensor

```
1. VCC → +3.3V
2. GND → GND
3. DO  → GPIO 34
```

### Шаг 7: Проверка соединений

```
Контрольный список:
  ☐ Все датчики питаются от 3.3V (НЕ 5V!)
  ☐ Все GND подключены к общей земле
  ☐ 4 резистора 10kΩ для DHT11 (pull-up)
  ☐ 3 резистора 10kΩ для Soil (делитель)
  ☐ GPIO 35 подключен через делитель (не напрямую!)
  ☐ Нет коротких замыканий между +3.3V и GND
```

## 🧪 Тестирование после сборки

### 1. Проверка питания

```bash
# Подключи USB, открой Serial Monitor (115200 бод)
pio device monitor

# Должно появиться:
[WiFi] AgriMap-Setup запущена
[DHT] Инициализация 4 датчиков...
```

### 2. Проверка DHT11

```
Подуй на каждый DHT11 — температура должна измениться на 1-2°C
Если датчик показывает -1.5°C → не подключен или нет pull-up резистора
```

### 3. Проверка Soil Sensor

```
Serial Monitor покажет:
  [SOIL] raw=1545  V=1.25V → 0% (dry=1550 wet=500)

Опусти датчик в стакан воды:
  [SOIL] raw=520   V=0.42V → 98%

Если raw=4095 → проблема с делителем или GPIO 35
```

### 4. Проверка Rain Sensor

```
Капни водой на датчик:
  Дождь: да

Вытри датчик:
  Дождь: нет

Если всегда "да" или "нет" → проверь GPIO 34 и DO пин
```

## 🐛 Типичные проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| DHT11 показывает -1.5°C | Нет pull-up резистора | Добавь 10kΩ между DATA и 3.3V |
| Soil sensor raw=4095 | Нет делителя или обрыв | Проверь 3 резистора делителя |
| ESP32 перезагружается | Короткое замыкание | Проверь +3.3V и GND на замыкание |
| Rain sensor всегда LOW | Датчик мокрый | Вытри датчик или проверь DO пин |
| WiFi не подключается | Неверный SSID/пароль | Удерживай BOOT при старте для сброса |
| Нет данных на сервере | Неверный IP сервера | Проверь SERVER_IP в настройках WiFi |
